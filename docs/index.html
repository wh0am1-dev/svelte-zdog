üé¨ animate

<script>
  import { Illustration, Ellipse } from 'svelte-zdog'
</script>

<Illustration
  element="svg"
  width={240}
  height={240}
  dragRotate
  update={node => ms => {
    node.rotate.y += 0.01
  }}
>
  <Ellipse diameter={128} stroke={32} color="#fbbf24" />
</Illustration>

primitive components accept an update function, node is the primitive, ms is millis
